Detailed Description

Biblioteca das funções.

Este ficheiro contém as 6 funções que são chamadas na main (test_app.c), sendo a sua função:
1º- Criar e inicializar o módulo
2º- Inserir elementos no módulo criado
3º- Identificar o valor máximo contido no módulo
4º- Identificar o valor mínimo contido no módulo
5º- Calcular a média (arredondada às unidades) dos valores do módulo
6º- Verificar o número de vezes que um certo valor foi inserido no módulo
